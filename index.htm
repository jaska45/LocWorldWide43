Localized Web Applications

This is a landing page for localized and multilingual web application samples.

Language specific URLs

You create a language specific pages for each language. There is a separate URL for each language.

English Finnish German French Japanese
www.sample.com www.sample.com/fi www.sample.com/de www.sample.com/fr www.sample.com/ja

Single URL ASP.NET Core Application

This uses ASP.NET Core that is a server side framework. You compile and deploy the application once, but the application selects the language on runtime based on the Accept-Language HTTP header, location or cookie.

www.aspsample.com

Single URL Angular Application

This uses Angular that is a a client side framework. You compile and deploy the application once, but the application selects the language on runtime. based on the Accept-Language HTTP header, location or the saved language.

www.angularsample.com
